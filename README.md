# API by TucanoWeb with NodeJS

Project made for one of my students, using NodeJS.



# Get Starter



## Install Dependencies

In the root of the project, run the command:

```bash
npm install
```



## Run Project

Yet in the root of the project, run the command:

```bash
nodemon .\app.js
```
